# Dualsystem
Simple C# WindowsForms program for the binary system

Hotkeys:
 - ESC = Exit Application
 - CTRL + R = Reset checkboxes
 - CTRL + A = Check all checkboxes
 - CTRL + P = Show result as MessageBox
 - CTRL + C = Copy result to Clipboard
 - CTRL + H = Show Hotkeys
